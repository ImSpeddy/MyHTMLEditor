# Shortcuts

---

## Shortcut list

| Shortcut           | Action                              |
| ------------------ | ----------------------------------- |
| `Ctrl + N`         | Create File                         |
| `Ctrl + O`         | Open File                           |
| `Ctrl + S`         | Save File                           |
| `Ctrl + Q`         | Close File                          |
| `Ctrl + W`         | Open Viewer                         |
| `Ctrl + Shift + W` | Open Viewer (based on current file) |
| `Ctrl + R`         | Refresh Viewer                      |
