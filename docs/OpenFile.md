# Open File

### Shortcut `Ctrl + O`

---

To open files, WebBox will prompt you to choose a file that matches the following file types:

- **HTML** (.html, .htm)
- **JavaScript** (.js, .jsx, .mjs, .cjs)
- **TypeScript** (.ts, .tsx)
- **CSS** (.css)
- **JSON** (.JSON)
- **Markdown** (.md)
- **PHP** (.php)
- **Text Documents** (.txt)
- **XML** (.xml)
